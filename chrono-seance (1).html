Rechercher sur YouTube
Sans clé : la recherche s'ouvre dans un nouvel onglet ; copiez ensuite le lien de la vidéo dans le champ « Lien direct ».
Avec une clé API YouTube Data v3 (gratuite, via Google Cloud Console), les résultats s'affichent ici, miniatures comprises — un clic remplit le lien.
En cas d'erreur 403 : retirez toute restriction « référents HTTP » sur la clé, un fichier ouvert localement n'en envoie pas. La clé reste sur cet appareil, elle n'est jamais incluse dans vos exports.
